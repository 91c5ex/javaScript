Scope

JavaScript variable are untyped: you can assign value of any type to a variable and you can later assign a value of different type to same variable .Varible are delcared with var keyword. JavaScript uses lexical scoping . Variable declared outside the function are global variable and are visible everywhere in the JavaScript Program. Variable delcared inside the function have function scope and are visible only inside the function.
