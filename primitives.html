Numbers.

JavaScript doesnot make and distinction between the integer type and the floating type.All number in javascript are treated as floating-point. JavaScript represent number using the 64 bit floating point number.Certain operations (such as arrayIndexing and bitwise operation) in javascript are performed as 32 bit operators

When number appear directly in javascript it's called a numeric literal

Floating-Point Literal

Floating-Point Literal can have a decimal point. They uses the synatx of the real number. A real value is represented as the integral part of number, followed by an decimal part and the frational part of the number.

Floating point literal may also be represented using the exponential notation. a real number followed by the letter e(or E), followed by option plus and minus

Arithmetic in JavaScript

JavaScript works with the number using the Arithmetic operator the language provide.These include

  • - for substraction
  • * for multiplication
  • / for division
  • % for module

Math Object provide more complex math operations. Underflow occur When the result of a numeric operation is closer to zero than the smallest representable number. In this case JavaScript return 0.
